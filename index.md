## Usage

* Install packages: `npm i`
* Install typings: `npm run typings`
* Build, Run & Watch: `npm start`

## Main Gulp Tasks
 * Default (Start App http://:localhost:8080/)
 * Spec (run Jasmine http://localhost:9002/)
 * E2e (Run Protractor http://:localhost:8080/)
